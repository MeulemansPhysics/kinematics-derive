Kinematics: Deriving Expressions

An interactive practice tool for AP Physics students learning to derive algebraic expressions from kinematic scenarios.

What it covers

18 scenarios across four categories: uniform acceleration, constant velocity, free fall, and two-body problems. Each scenario asks students to derive expressions using the three kinematic equations — the same skill tested on AP free-response questions.

How to use it

Choose a scenario category using the filter buttons, then read the scenario and derive the requested expression. You can either drag symbols from the palette into the answer box, or switch to Type mode and enter your expression with the keyboard. The checker is algebraically flexible — it recognizes equivalent forms regardless of order, and adjacent symbols are treated as multiplication so no × is needed.
Hints are available at two levels: "Show given quantities" reveals the knowns, "Which equation applies?" nudges toward the right starting point, and the Algebra hint button gives a more direct step-by-step nudge.
